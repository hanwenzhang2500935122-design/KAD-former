## 三类病害的关键诊断链

**黑星病**: 苹果 → 黑星病 → 橄榄色绒毛斑（叶片正面/初期） → 深色凹陷病斑（中期） → 叶片卷曲+早期落叶（晚期）

**黑腐病**: 苹果 → 黑腐病 → 紫色小点（初期） → 蛙眼状轮纹斑（中期） → 分生孢子器+果实僵化（晚期）

**锈病**: 苹果 → 锈病 → 黄绿色小斑（初期） → 红色星状外缘+橙黄色锈子器（中期） → 组织增厚+枝条菌瘿（晚期）
